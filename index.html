node-virtual-folder
(API Documentation)

About node-virtual-folder

An updatable virtual folder tree structure created with Node, Express, and MongoDB. It can read, post, and delete virtual folders and stores them in the cloud.

Methods

GET POST DELETE

endpoint: /folders

Get single folder

GET /folders/:id
> Returns a json response with the folder object. 
{
	"children":[],
	"_id":"60a6d0015d87f58c1f872a74",
	"name":"Folder 2",
	"root":false,
	"parent_id":"60a37f4517d49a7a1d97271f",
	"__v":0
}

Add(Post) folder

POST /folders

body: { name: folder_name, parent_id: parent_id }
Using axios: 
axios.post('https://node-virtual-folder.herokuapp.com/folders', {
	name: String(name),
	parent_id: String(parent_id)
})
.then(result => console.log(result) )
.catch(error => console.log(error) );

Delete a folder

DELETE /folders

body: { id: _id }
Using axios: 
axios.delete('https://node-virtual-folder.herokuapp.com/folders', {
	data: { id: String(_id) }
})
.then(result => console.log(result) )
.catch((error) => console.log(error) );
